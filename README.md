# Outlook 群發郵件工具

一個專業的 Windows 桌面應用程式，用於從 Excel 文件或剪貼簿讀取收件人資訊，透過 Microsoft Outlook 發送個性化郵件。每封郵件單獨發送，確保收件人隱私。

## 🌟 主要功能

### 📊 多種資料來源支援
- **Excel 文件讀取**：支援 `.xlsx` 和 `.xls` 格式
- **剪貼簿資料**：直接從複製的表格資料讀取收件人
- **智慧識別**：自動識別姓名和郵件地址欄位

### 📧 郵件功能
- **個性化內容**：支援 `{姓名}` 變數自動替換
- **附件支援**：可添加多個附件
- **郵件預覽**：發送前可預覽郵件內容
- **隱私保護**：每位收件人收到獨立郵件，不會看到其他收件人

### 🔧 操作體驗
- **進度顯示**：即時顯示發送進度
- **錯誤處理**：詳細的錯誤報告和處理
- **中途停止**：可隨時停止發送過程
- **狀態回饋**：清楚的操作狀態提示

## 🚀 快速開始

### 系統需求
- **作業系統**：Windows 10/11
- **必須軟體**：Microsoft Outlook (桌面版)
- **Python版本**：Python 3.8 或更高版本

### 安裝步驟

1. **克隆專案**
   ```bash
   git clone <repository-url>
   cd outlook-mail-sender
   ```

2. **安裝相依套件**
   ```bash
   pip install -r requirements.txt
   ```

3. **運行應用程式**
   ```bash
   python outlook_mail_sender.py
   ```

## 📖 使用教學

### 1. 準備收件人資料

#### 方法一：使用 Excel 文件
創建 Excel 文件，包含以下欄位：
```
姓名        郵件
張三        zhangsan@company.com
李四        lisi@company.com
王五        wangwu@company.com
```

**支援的欄位名稱**：
- 姓名欄：`姓名`、`Name`、`名字`、`名稱`、`联系人`、`聯絡人`
- 郵件欄：`郵件`、`Email`、`E-Mail`、`電子郵件`、`电子邮件`、`邮箱`

#### 方法二：使用剪貼簿
1. 在 Excel 或任何表格軟體中選擇包含收件人資訊的區域
2. 複製到剪貼簿 (Ctrl+C)
3. 在應用程式中點擊「從剪貼簿貼上」

### 2. 編寫郵件內容

#### 主題設定
在「主題」欄位輸入郵件標題。

#### 內容編寫
在「內容」區域編寫郵件內容，支援以下變數：
- `{姓名}` - 自動替換為收件人姓名
- `{name}` - 同上（英文）
- `{Name}` - 首字母大寫版本
- `{NAME}` - 全大寫版本

**範例**：
```
親愛的 {姓名}，

您好！這是一封重要通知。

請查收附件中的相關資料。

祝好！
```

### 3. 添加附件（可選）
1. 勾選「包含附件」選項
2. 點擊「選擇附件」按鈕
3. 選擇要附加的文件

### 4. 預覽和發送
1. 點擊「預覽郵件」查看第一封郵件的實際內容
2. 確認無誤後點擊「發送郵件」
3. 確認發送對話框
4. 監控發送進度

## 🛠️ 進階功能

### 自動欄位識別
程式會自動識別 Excel 文件中的欄位：
- 根據欄位標題自動匹配
- 根據內容特徵判斷（如包含 @ 符號的欄位作為郵件）
- 支援中英文欄位名稱

### 錯誤處理
- **無效郵件地址**：自動跳過格式錯誤的郵件地址
- **發送失敗**：記錄失敗原因，繼續發送其他郵件
- **Outlook 連接問題**：提供詳細錯誤訊息

### 發送控制
- **進度追蹤**：即時顯示發送進度和當前收件人
- **暫停/停止**：可隨時中斷發送過程
- **發送延遲**：自動在郵件之間添加短暫延遲，避免被識別為垃圾郵件

## 📋 支援的資料格式

### Excel 文件格式
```
| 姓名 | 郵件地址 |
|------|----------|
| 張三 | zhang@example.com |
| 李四 | li@example.com |
```

### 剪貼簿格式
支援以下分隔符：
- Tab 分隔（從 Excel 複製時的預設格式）
- 逗號分隔 (CSV 格式)
- 分號分隔
- 豎線分隔

## ⚠️ 注意事項

### 安全性
- 本工具通過 Microsoft Outlook 發送郵件，使用您的 Outlook 帳戶
- 請確保您有發送郵件的適當權限
- 建議先小批量測試，確認無誤後再大量發送

### 效能考量
- 大量郵件發送可能需要較長時間
- 建議單次發送不超過 500 封郵件
- 如需發送大量郵件，可分批進行

### Outlook 設定
- 確保 Outlook 已正常安裝並配置
- 建議在發送前先測試 Outlook 是否能正常發送郵件
- 某些企業環境可能對 COM 物件有限制

## 🐛 問題排解

### 常見問題

**Q: 提示 "win32com.client 未安裝"**
A: 執行 `pip install pywin32` 安裝相關套件

**Q: 無法連接到 Outlook**
A: 
- 確認 Outlook 已安裝並已設定帳戶
- 嘗試先手動開啟 Outlook
- 檢查是否有防毒軟體阻擋

**Q: Excel 讀取失敗**
A:
- 確認文件格式正確（.xlsx 或 .xls）
- 檢查檔案是否被其他程式開啟
- 確認檔案路徑無中文或特殊字元

**Q: 剪貼簿資料無法識別**
A:
- 確認複製的資料包含有效的郵件地址
- 嘗試在 Excel 中整理資料後再複製
- 檢查資料格式是否正確
