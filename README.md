# Outlook 群發郵件工具

一個功能完整的 Windows 桌面應用程式，可以從 Excel 文件或剪貼簿中提取收件人資訊，並透過 Microsoft Outlook 發送個性化郵件。

## 🎯 主要功能

- **多種收件人來源**：支援從 Excel 文件（.xlsx/.xls）或剪貼簿讀取收件人
- **智能收件人識別**：自動識別姓名和郵件地址欄位
- **個性化郵件**：支援變數替換（如 `{姓名}`）
- **隱私保護**：每封郵件獨立發送，收件人無法看到其他收件人
- **附件支援**：可添加單個或多個附件
- **郵件預覽**：發送前可預覽郵件內容
- **進度追蹤**：實時顯示發送進度和狀態
- **錯誤處理**：完善的錯誤處理和日誌記錄

## 📋 系統需求

- **作業系統**：Windows 10/11
- **Python**：3.8 或更高版本
- **Microsoft Outlook**：已安裝並配置的桌面版 Outlook
- **權限**：需要允許 Python 訪問 Outlook COM 介面

## 🚀 安裝與使用

### 1. 安裝依賴套件

```bash
pip install -r requirements.txt
```

### 2. 運行程式

```bash
python outlook_bulk_email_tool.py
```

### 3. 使用步驟

1. **載入收件人**：
   - 點擊「選擇 Excel 文件」選擇包含收件人資訊的 Excel 文件
   - 或點擊「從剪貼簿貼上」讀取複製的表格資料

2. **設置郵件內容**：
   - 輸入郵件主題
   - 編寫郵件內容（可使用 `{姓名}` 變數進行個性化）

3. **選擇附件**（可選）：
   - 勾選「包含附件」
   - 點擊「選擇附件」選擇要附加的文件

4. **預覽和發送**：
   - 點擊「預覽郵件」查看第一封郵件的實際內容
   - 點擊「發送郵件」開始群發

## 📊 支援的資料格式

### Excel 文件格式

支援以下欄位名稱（不區分大小寫）：
- **姓名欄位**：姓名、name、Name、名字、收件人、recipient
- **郵件欄位**：郵件、email、Email、電子郵件、e-mail、E-Mail、mail、Mail

範例：
```
姓名        郵件
張三        zhangsan@company.com
李四        lisi@company.com
王五        wangwu@company.com
```

### 剪貼簿格式

支援從 Excel 或其他表格軟體複製的文字，以 tab、逗號或分號分隔：

```
姓名	郵件
張三	zhangsan@company.com
李四	lisi@company.com
```

或：
```
姓名,郵件
張三,zhangsan@company.com
李四,lisi@company.com
```

## 🔧 進階功能

### 個性化變數

在郵件內容中使用 `{姓名}` 變數，程式會自動替換為對應的收件人姓名。如果沒有姓名，會自動替換為「您好」。

範例：
```
親愛的 {姓名}，

這是您的專屬通知信。

祝好！
```

### 附件支援

- 支援添加多個附件
- 支援各種文件格式
- 自動檢查文件是否存在

## ⚠️ 注意事項

1. **Outlook 權限**：首次使用時，Outlook 可能會詢問是否允許程式訪問，請選擇「允許」
2. **發送限制**：建議不要一次性發送過多郵件，避免被標記為垃圾郵件
3. **網路連接**：確保網路連接正常，Outlook 可以正常發送郵件
4. **收件人驗證**：程式會自動驗證郵件地址格式，無效的地址會被跳過

## 🐛 故障排除

### 常見問題

1. **無法連接 Outlook**
   - 確保已安裝 Microsoft Outlook 桌面版
   - 檢查 Outlook 是否正常運行
   - 嘗試重新啟動 Outlook

2. **Excel 文件讀取失敗**
   - 確保文件格式為 .xlsx 或 .xls
   - 檢查文件是否被其他程式佔用
   - 確保文件包含有效的郵件地址

3. **郵件發送失敗**
   - 檢查網路連接
   - 確認 Outlook 帳戶配置正確
   - 檢查收件人郵件地址是否有效
