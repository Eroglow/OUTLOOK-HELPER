# Excel/剪贴板群发邮件自动化工具

一个专业的Windows桌面应用程序，用于通过Excel文件或剪贴板数据批量发送个性化邮件至Outlook。

## 🌟 主要特性

- **可视化操作界面** - 基于CustomTkinter的现代化GUI
- **多种数据输入方式** - 支持Excel文件、CSV文件和剪贴板数据
- **智能字段映射** - 自动检测邮箱和姓名字段
- **个性化邮件模板** - 支持变量替换，如 `{姓名}`、`{电子邮箱}`
- **一对一发送** - 每位收件人独立接收，互不可见
- **实时进度跟踪** - 显示发送进度和统计信息
- **详细日志记录** - 完整的操作日志和错误信息
- **Outlook集成** - 直接通过本地Outlook发送邮件

## 📋 系统要求

- Windows 10 或更高版本
- Python 3.8 或更高版本
- Microsoft Outlook 桌面应用程序（已安装并配置邮箱）

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动程序

**方法一：使用启动脚本**
```bash
start.bat
```

**方法二：直接运行**
```bash
python email_automation.py
```

## 📖 使用说明

### 数据导入

1. **Excel/CSV文件导入**
   - 点击"选择Excel/CSV文件"按钮
   - 支持 `.xlsx`、`.xls`、`.csv` 格式
   - 自动处理多种编码格式

2. **剪贴板导入**
   - 在Excel中选择数据并复制（Ctrl+C）
   - 点击"从剪贴板导入"按钮
   - 程序将自动解析表格数据

### 字段映射

- 程序会自动检测可能的邮箱和姓名字段
- 如需调整，可通过下拉菜单手动选择
- 确保邮箱字段包含有效的电子邮箱地址

### 邮件模板

1. **邮件主题** - 输入邮件的标题
2. **邮件内容** - 编写邮件正文
3. **变量支持** - 使用 `{字段名}` 格式插入动态内容

**常用变量示例：**
```
亲爱的 {姓名}，

感谢您的参与！您的邮箱 {电子邮箱} 已收到此确认邮件。

此致
敬礼
```

### 发送邮件

1. 点击"预览邮件"查看邮件样本
2. 点击"开始发送"启动批量发送
3. 实时查看发送进度和日志信息
4. 如需停止，点击"停止发送"按钮

## 📝 注意事项

1. **Outlook要求**
   - 必须安装Microsoft Outlook桌面应用
   - 需要配置至少一个邮箱账户
   - 首次运行可能需要授权COM组件访问

2. **安全提醒**
   - 首次发送时Outlook可能显示安全警告
   - 需要手动点击"允许"继续操作
   - 建议先用小批量数据测试

3. **性能优化**
   - 程序自动控制发送速度（每封邮件间隔1秒）
   - 避免被邮件服务器限制
   - 大批量发送建议分批处理

## 📄 日志文件

程序运行时会自动生成 `email_automation.log` 文件，记录：
- 程序启动和关闭时间
- 数据导入操作
- 邮件发送状态
- 错误信息和异常

## 🔧 故障排除

### 常见问题

1. **无法连接Outlook**
   - 确保Outlook已正确安装
   - 检查是否配置了邮箱账户
   - 重启Outlook应用程序

2. **文件读取失败**
   - 检查文件格式是否支持
   - 确认文件未被其他程序占用
   - 尝试使用UTF-8编码保存CSV文件

3. **邮件发送失败**
   - 验证收件人邮箱格式
   - 检查网络连接状态
   - 查看Outlook发送规则设置

### 技术支持

如遇到问题，请检查以下内容：
1. 系统日志文件 `email_automation.log`
2. Outlook错误提示信息
3. Python环境和依赖包版本
