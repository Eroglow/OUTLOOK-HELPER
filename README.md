# 📧 Outlook 群發郵件工具

一個簡單易用的 Windows 桌面應用程式，可從 Excel 文件或剪貼簿中提取收件人資訊，並透過 Microsoft Outlook 發送個性化郵件。

## ✨ 功能特點

- 📁 **多種資料來源**：支援從 Excel 文件（.xlsx/.xls）或剪貼簿讀取收件人資料
- 👤 **個性化郵件**：支援使用變數（如 `{姓名}`）來個性化每封郵件
- 🔒 **隱私保護**：每位收件人收到獨立郵件，不會看到其他收件人
- 📎 **附件支援**：可添加單個或多個附件
- 👁️ **郵件預覽**：發送前可預覽郵件內容
- 📊 **進度追蹤**：即時顯示發送進度
- 🎨 **友善介面**：直覺的圖形化操作介面

## 🔧 系統需求

- Windows 7/8/10/11
- Python 3.8 或更高版本
- Microsoft Outlook 桌面應用程式（已安裝並登入）

## 📦 安裝步驟

### 方法一：從原始碼執行

1. **克隆或下載專案**
   ```bash
   git clone <repository-url>
   cd outlook-bulk-mailer
   ```

2. **安裝 Python 依賴套件**
   ```bash
   pip install -r requirements.txt
   ```

3. **執行程式**
   ```bash
   python outlook_bulk_mailer.py
   ```

### 方法二：使用打包的 EXE 檔案（建議）

如果您想要建立獨立的 .exe 檔案，無需安裝 Python：

1. **安裝 PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **打包成 EXE**
   ```bash
   pyinstaller --onefile --windowed --icon=icon.ico outlook_bulk_mailer.py
   ```

3. **找到生成的 EXE 檔案**
   - 檔案位於 `dist/outlook_bulk_mailer.exe`
   - 可以將此檔案複製到任何 Windows 電腦上直接執行

## 📖 使用說明

### 1. 準備收件人資料

#### Excel 格式範例：
| 姓名 | 郵件 |
|------|------|
| 張三 | zhangsan@company.com |
| 李四 | lisi@company.com |
| 王五 | wangwu@company.com |

支援的欄位名稱：
- 姓名：`姓名`、`Name`、`名稱`、`名字`、`收件人`
- 郵件：`郵件`、`Email`、`Mail`、`E-mail`、`電子郵件`、`信箱`

#### 剪貼簿格式：
從 Excel 複製表格資料，或使用 Tab/逗號分隔的文字格式。

### 2. 操作流程

1. **載入收件人資料**
   - 點擊「選擇 Excel 文件」選擇 Excel 檔案
   - 或從 Excel/表格複製資料後，點擊「從剪貼簿貼上」

2. **編寫郵件內容**
   - 輸入郵件主題
   - 編寫郵件內容（可使用 `{姓名}` 變數）
   - 如需附件，勾選「包含附件」並選擇檔案

3. **預覽確認**
   - 點擊「預覽郵件」查看第一封郵件的實際內容
   - 確認格式和內容正確

4. **發送郵件**
   - 點擊「發送郵件」
   - 確認發送數量
   - 等待發送完成

## 🎯 使用範例

### 個性化郵件內容範例：

```
親愛的 {姓名}，

感謝您參加我們的活動！

這是您的專屬邀請碼：ABC123

期待您的蒞臨！

祝好
活動團隊
```

每位收件人將看到自己的姓名，例如：
- 張三會看到「親愛的張三」
- 李四會看到「親愛的李四」

## ⚠️ 注意事項

1. **Outlook 設定**
   - 確保 Microsoft Outlook 已安裝並登入帳號
   - Outlook 需要在背景運行

2. **測試建議**
   - 首次使用建議先發送給自己測試
   - 可以修改程式碼中的 `mail.Send()` 為 `mail.Display()` 來預覽而不發送

3. **發送限制**
   - 注意郵件服務商的發送限制
   - 建議分批發送大量郵件

4. **錯誤處理**
   - 發送失敗的郵件會記錄在錯誤日誌檔案中
   - 檔名格式：`email_error_log_YYYYMMDD_HHMMSS.txt`

## 🛠️ 疑難排解

### 問題：無法連接 Outlook
**解決方案**：
1. 確認 Outlook 已安裝
2. 確認 Outlook 已登入帳號
3. 嘗試手動開啟 Outlook
4. 檢查防毒軟體是否阻擋

### 問題：無法讀取 Excel
**解決方案**：
1. 確認 Excel 檔案格式正確（.xlsx 或 .xls）
2. 確認檔案未被其他程式開啟
3. 檢查欄位名稱是否符合預期

### 問題：郵件發送失敗
**解決方案**：
1. 檢查收件人郵件地址格式
2. 確認網路連接正常
3. 檢查 Outlook 帳號設定
4. 查看錯誤日誌檔案
