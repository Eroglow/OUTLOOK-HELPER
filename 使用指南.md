# 📧 Excel/剪贴板群发邮件自动化工具 - 使用指南

## 🎯 快速开始 3 步骤

### 第 1 步：准备数据
1. **方式一：Excel/CSV文件**
   - 准备包含邮箱地址的Excel或CSV文件
   - 确保至少包含"邮箱"和"姓名"两列
   - 点击"选择Excel/CSV文件"按钮导入

2. **方式二：剪贴板导入**
   - 在Excel中选择数据范围（包含标题行）
   - 按 Ctrl+C 复制数据
   - 在程序中点击"从剪贴板导入"

### 第 2 步：配置邮件
1. **字段映射**
   - 程序会自动检测邮箱和姓名字段
   - 如需调整，使用下拉框重新选择

2. **编写邮件**
   - 输入邮件主题
   - 编写邮件内容，可使用变量如 `{姓名}`, `{电子邮箱}`
   - 可选择预设模板：商务邮件、感谢邮件、邀请邮件
   - 可选择添加附件

### 第 3 步：发送邮件
1. 点击"预览邮件"查看效果
2. 点击"开始发送"执行群发
3. 观察发送进度和日志信息
4. 如需停止，点击"停止发送"

## 📊 数据格式示例

### Excel/CSV文件格式
```
姓名      | 电子邮箱           | 公司        | 职位
---------|-------------------|------------|--------
张小明    | ming@abc.com      | ABC科技     | 工程师
李小华    | hua@xyz.com       | XYZ公司     | 经理
```

### 剪贴板数据格式
- 从Excel复制时确保包含标题行
- 支持制表符分隔的数据
- 程序会自动解析表格结构

## 🎨 邮件模板变量

### 基础变量
- `{姓名}` - 替换为数据中的姓名字段
- `{电子邮箱}` - 替换为收件人邮箱地址

### 自定义变量
- `{公司}` - 如果数据中有"公司"列
- `{职位}` - 如果数据中有"职位"列
- `{任何列名}` - 可使用任何数据列名作为变量

### 变量使用示例
```
尊敬的 {姓名}，

您好！我是来自 [您的公司] 的客服。

根据您在 {公司} 担任 {职位} 的工作经历，我们为您准备了专业的解决方案。

如有疑问，请联系 {电子邮箱}。

谢谢！
```

## 🛠️ 高级功能

### 1. 邮件模板
- **商务邮件**：正式的商务沟通模板
- **感谢邮件**：适用于活动感谢、客户关怀
- **邀请邮件**：会议、活动邀请模板

### 2. 附件支持
- 支持任何格式的附件文件

- 图片：JPG、PNG、GIF等
- 压缩包：ZIP、RAR等

### 3. 发送控制
- 自动控制发送间隔（2秒/封）
- 支持暂停和继续发送
- 实时显示发送进度
- 详细的成功/失败统计

## ⚠️ 重要注意事项

### 系统要求
- ✅ Windows 10 或更高版本
- ✅ 已安装并配置 Microsoft Outlook
- ✅ Python 3.8+ 环境

### 首次使用
1. **Outlook安全提醒**
   - 首次发送时会出现安全警告
   - 点击"允许"继续操作
   - 这是正常的COM组件访问提醒

2. **账户配置**
   - 确保Outlook中至少配置一个邮箱账户
   - 账户需要处于正常登录状态
   - 建议先发送测试邮件验证配置

### 最佳实践
1. **小批量测试**
   - 首次使用建议先用5-10个邮箱测试
   - 确认邮件格式和内容无误后再大批量发送

2. **数据备份**
   - 发送前请备份原始数据
   - 重要邮件建议分批次发送

3. **合规性**
   - 确保收件人同意接收邮件
   - 遵守反垃圾邮件法规
   - 提供退订选项（如需要）

## 🚨 故障排除

### 常见问题及解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无法连接Outlook | Outlook未启动或未配置账户 | 启动Outlook并确认账户正常 |
| 文件读取失败 | 文件格式不支持或文件损坏 | 检查文件格式，尝试重新保存 |
| 邮箱格式错误 | 数据中包含无效邮箱格式 | 检查并修正邮箱格式 |
| 发送速度慢 | 网络延迟或邮件服务器限制 | 正常现象，耐心等待 |
| 部分邮件发送失败 | 邮箱不存在或服务器拒绝 | 查看日志了解具体失败原因 |

### 调试信息
- 所有操作都会记录在 `email_automation.log` 文件中
- 发送过程中的错误信息会显示在日志区域
- 遇到问题时可查看日志文件获取详细信息

## 📈 发送统计

程序会实时显示：
- **总邮件数**：待发送的邮件总数
- **成功数量**：已成功发送的邮件数
- **失败数量**：发送失败的邮件数
- **处理进度**：当前处理百分比

## 💡 使用技巧

1. **邮件主题优化**
   - 使用变量让主题更个性化
   - 例如：`{姓名}，您的专属方案已准备就绪`

2. **内容格式化**
   - 使用适当的段落分隔
   - 重要信息可以使用符号标记（如📧、🎯）

3. **变量组合**
   - 可以在同一句话中使用多个变量
   - 例如：`尊敬的 {姓名}，感谢 {公司} 对我们的信任`

4. **附件管理**
   - 大文件建议压缩后添加
   - 文件名使用中文时确保编码正确
